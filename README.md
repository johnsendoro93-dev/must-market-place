## 🌟 Overview
The **MUST Campus Marketplace** is a digital platform designed to bridge the gap between student entrepreneurs and the campus community. This project provides a centralized hub for buying, selling, and trading within the MUST ecosystem.

## ✨ Features
* **Student-Centric Listings:** Easy navigation for campus-specific products.
* **Responsive Design:** Optimized for mobile use on campus.
* **Dynamic Search:** Quick filtering of items and services.
* **Community Focused:** Built by students, for students.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom MUST-themed styling)
* **Interactivity:** Vanilla JavaScript
* **Deployment:** Vercel
* **Version Control:** GitHub
